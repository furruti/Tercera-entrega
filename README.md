# Mi Proyecto Django

## Requisitos
- Python 3
- Django

## Instrucciones para ejecutar
1. Clonar el repositorio
2. Crear un entorno virtual
3. Instalar dependencias con `pip install -r requirements.txt`
4. Ejecutar el servidor con `python manage.py runserver`
